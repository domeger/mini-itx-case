# mini-itx-case
A simple DIY mini ITX case for the [Turing Pi](https://turingpi.com)

To assemble the case you will need: 
1. the case parts
2. 8 x M3 10mm MF standoffs 
3. 4 x M3 30mm FF standoffs
4. 4 x M3 30mm screws

Material: Acrylic 4mm
